# Fake Tweet Detection Using Machine Learning

## Overview

This project is about detecting whether a tweet is fake using machine learning. It uses something called Natural Language Processing and a Naïve Bayes classifier to do this. The project also has an interface that people can use to get predictions right away.

## Features

* The project cleans up the tweets to make them easier to work with

* It uses techniques to figure out what is important in the tweets

* The project can tell if a tweet is fake or real using the Naïve Bayes classifier

* It has an interface that people can use to interact with the project

* The project can make predictions in time

## Technologies Used

* The project uses Python to make it work

* It uses Pandas to help with data

* NumPy is used for math and numbers

* Scikit-learn is used for machine learning

* The project uses Natural Language Processing to understand the tweets

* Tkinter is used to make the interface

## Dataset

The project uses many datasets that have real and fake tweets to train the model. 
These datasets are not included here because they are too big.

## Model

* The project uses a Naïve Bayes Classifier to make predictions

## How to Run

1. First you need to install the things the project needs to work. You can do this by running the following command in your terminal:

```bash

pip install -r requirements.txt

```

2. Then you can run the project by running the following command:

```bash

python app.py

```

## Output

The project will tell you if the tweet you entered is fake or real using the machine learning model it was trained on.
